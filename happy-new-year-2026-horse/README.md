# Happy New Year 2026 • Year of the Horse

Mini web-app phong cách “đếm ngược → thiệp Tết” (bản demo mặt hình thức).

## Chạy thử
- Mở `index.html` (hoặc dùng Live Server trong VS Code).

## Luồng
- Trước giờ: hiện **countdown** (thiệp bị khóa).
- Đúng giờ: tự mở **thiệp** (ngay trong `index.html`), hiện màn **bốc quẻ** rồi mở lời chúc ngẫu nhiên.

## Assets
- `assets/horse.png`: mascot ngựa (theo ảnh bạn cung cấp)
- `assets/fortune_box.png`: hộp thẻ bốc quẻ (đã xử lý nền trong suốt)
